# README #


### What is this repository for? ###

* An IOS questionnaire 


### How do I get set up? ###

You need 

* XCODE 7
* [CocoaPods](https://cocoapods.org/)

pro tip: be sure to use `git clone --recursive` to get the submodules

```
run pod install
open workaday.xcworkspace in XCode 
```
